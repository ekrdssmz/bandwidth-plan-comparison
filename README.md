## 搬瓦工 VPS 带宽是什么意思？共享还是独享？

先把这个问题解决掉，不然后面的数字全是误解。

搬瓦工标注的带宽（1Gbps、2.5Gbps、10Gbps）是**上行端口峰值速度**，不是你独享的带宽。说白了，这个带宽是你和同节点的其他用户共享的。

共享听起来不好听，但实际影响不大——因为大部分时候，用户的使用时间并不完全重叠。如果你一个人在跑大文件传输，确实能跑接近峰值的速度。

真正影响实际速度的，是**线路质量**，不是标称带宽的数字。一条 1Gbps 的 CN2 GIA 专线，比一条 10Gbps 的普通 163 骨干网，在晚高峰的实际表现要稳定得多。后面会解释为什么。

还有一点要知道：流量用完了，VPS 会自动暂停到下个月，**不会超量计费**。这个机制反而省心——不用担心某个月跑多了账单爆掉。

## 关于流量，有几个常见误区

**流量超了会怎样？**

当月流量用完，VPS 自动暂停，等到下个月 1 号自动恢复。不额外收费，也不强制升级。有的机器还能在 KiwiVM 后台点"临时恢复"，用少量额外流量先恢复访问。

**1TB 流量实际能跑多久？**

取决于你传什么。跑 Web 服务，1TB 一个月很宽裕。跑视频分发就另说了。大流量需求建议看 E-Commerce 的高配套餐，最高有 12TB/月的版本。

**CN2 GIA 和普通 CN2 GT 的带宽体验差多少？**

CN2 GT 理论带宽也不小，但晚高峰经常出现丢包和拥堵，实际速度会掉。CN2 GIA 因为容量限制和高成本，保持的是较稳定的传输质量，高峰期丢包率明显低。如果你的业务时间段覆盖晚 8 点到 11 点，这个差别很明显。

## 常见问题

**Q：搬瓦工带宽是独享的吗？**

A：不是独享，是共享带宽。但标注带宽是你单台 VPS 可用的峰值速度，共享的是物理上行端口的总容量。日常使用基本感受不到竞争压力，高负载时才有影响。

**Q：月流量和带宽有什么关系？**

A：两个维度独立计算。带宽决定你**同一时刻**的传输速度上限，月流量决定你**一个月内**总共能传多少数据。跑满 1Gbps 带宽，1TB 流量大约 3 小时就用完了。日常 Web 服务当然不会这么跑。

**Q：E-Commerce 套餐和 Basic 套餐机房一样，为什么价格差这么多？**

A：带宽不一样（1Gbps vs 2.5Gbps 起步），可选的优化线路机房也不一样。E-Commerce 套餐才能选 DC6、DC9 等 CN2 GIA 机房，这几个机房在 Basic 套餐里选不到。

**Q：搬瓦工的带宽够跑什么应用？**

A：1Gbps 可以跑普通网站、VPN、代理、开发环境；2.5Gbps 适合中等流量的跨境电商、API 服务；10Gbps 是高配套餐给高流量业务用的，个人用户基本用不满。

**Q：限量版套餐经常缺货怎么办？**

A：搬瓦工的限量版套餐（比如各种 Box 系列）确实经常断货，官网库存页面可以查看实时状态。常规套餐（KVM、CN2 GIA-E 主力套餐）一般不缺货，下单基本即时开通。
